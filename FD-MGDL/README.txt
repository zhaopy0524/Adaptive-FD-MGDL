This is the FD-MGDL code for two- and three-dimensional Helmholtz problems.
Run the file 'main_run.py' to start first grade, and the file 'add_grade.py' to add grade.
For different Helmholtz equations, the corresponding finite difference discretizations are implemented in 'data_generate.py'.
