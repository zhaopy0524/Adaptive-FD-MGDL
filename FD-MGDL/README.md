In this work, we compare our proposed methods, FD-MGDL and FD-SGDL, with several baseline approaches: Mscale [1], FBPINN [2], SIREN [3], PINN [4], and Pre-PINN [5]. The source code for these comparative methods is publicly available at the following repositories:
- Mscale: https://github.com/xuzhiqin1990/mscalednn
- FBPINN: https://github.com/benmoseley/FBPINNs
- SIREN: https://github.com/vsitzmann/siren
- PINN: https://github.com/maziarraissi/PINNs
